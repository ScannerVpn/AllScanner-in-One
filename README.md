# 🛡️ VPN Scanner Suite

هشت اسکنر VPN در یک برنامه‌ی واحد با تب‌بندی بالای صفحه. با یک بار اجرا، همه‌ی اسکنرها بالا می‌آیند و می‌توانی بین آن‌ها جابجا شوی:

- **Surfshark**
- **NordVPN**
- **ExpressVPN**
- **PureVPN**
- **Mullvad**
- **PIA** (Private Internet Access)
- **Windscribe**
- **Proton VPN**

## اجرا

### روش ساده (ویندوز)
فایل `Start.bat` را دابل‌کلیک کن. وابستگی‌های لازم نصب می‌شوند، همه‌ی اسکنرها بالا می‌آیند و مرورگر به‌صورت خودکار باز می‌شود.

### روش دستی
```bash
git clone https://github.com/ScannerVpn/AllScanner-in-One.git
cd AllScanner-in-One
npm install        # فقط بار اول (express را برای دو اسکنر نصب می‌کند)
npm start
```
سپس مرورگر روی `http://localhost:8080/` باز می‌شود.

## ساختار

```
vpn-scanner-suite/
├── suite.js          # نقطه‌ی ورود واحد: هر ۴ اسکنر را بالا می‌آورد + سرور پوسته
├── shell.html        # صفحه‌ی تب‌دار که هر اسکنر را در iframe نشان می‌دهد
├── package.json
├── Start.bat         # اجرای یک‌کلیکی ویندوز
└── scanners/
    ├── surfshark/    # پورت 3002 (Node خام)
    ├── nord/         # پورت 3000 (Node خام)
    ├── expressvpn/   # پورت 3003 (Express)
    ├── purevpn/      # پورت 3004 (Express)
    ├── mullvad/      # پورت 3005 (Express)
    ├── pia/          # پورت 3006 (Express)
    ├── windscribe/   # پورت 3007 (Express)
    └── proton/       # پورت 3008 (Express)
```

## چطور کار می‌کند

- `suite.js` هر اسکنر را به‌صورت یک پروسه‌ی جداگانه روی پورت خودش اجرا می‌کند (کد اسکنرها دست‌نخورده می‌ماند).
- یک سرور سبک روی پورت **8080** صفحه‌ی `shell.html` را سرو می‌کند که شامل نوار تب و یک `iframe` برای هر اسکنر است.
- هر iframe داشبورد اصلی همان اسکنر را از پورت خودش بارگذاری می‌کند؛ بنابراین تمام قابلیت‌های هر اسکنر بدون تغییر کار می‌کنند.
- **بارگذاری تنبل:** هر اسکنر فقط وقتی اولین بار تبش را باز کنی لود می‌شود.

## پورت‌ها

| اسکنر | پورت |
|--------|------|
| پوسته (تب‌ها) | 8080 |
| NordVPN | 3000 |
| Surfshark | 3002 |
| ExpressVPN | 3003 |
| PureVPN | 3004 |
| Mullvad | 3005 |
| PIA | 3006 |
| Windscribe | 3007 |
| Proton VPN | 3008 |

اگر پورتی اشغال بود، می‌توانی پورت پوسته را تغییر بدهی:
```bash
SHELL_PORT=9000 npm start
```

## نکته‌ی مهم هنگام اسکن
برای نتیجه‌ی درست، قبل از اسکن، اپ VPN را کاملاً **خاموش** کن؛ وگرنه ترافیک از داخل تونل عبور می‌کند و همه‌ی سرورها اشتباهاً «در دسترس» نشان داده می‌شوند.
